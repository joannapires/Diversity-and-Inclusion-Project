# Diversity and Inclusion Project

## Project Overview

This project is a simple, responsive webpage designed to introduce the core concepts of **diversity and inclusion** in a workplace or educational context. It is aimed at users who are new to the topic and want clear, straightforward information presented in an organised and accessible way.

The site has been built using **HTML, CSS, and Bootstrap**, with a focus on clean layout, visual clarity, and ease of navigation.

---

## External User Goal

The external user wants to:
- Gain a basic understanding of diversity and inclusion
- Learn why these concepts are important in workplaces and educational environments
- Access simple tips and trusted resources
- Navigate content easily without prior knowledge of the topic

---

## Site Owner Goal

The site owner aims to:
- Create a visually appealing and easy-to-use webpage
- Clearly introduce diversity and inclusion concepts
- Demonstrate effective use of HTML and CSS
- Use Bootstrap components to structure and style the content
- Present information in a logical and accessible format

---

## Features

The website includes the following features:

### Hero Section
- A clear headline introducing diversity and inclusion
- A short supporting description
- A simple background colour or image
- Designed using Bootstrap’s layout utilities or a Jumbotron-style header

### Information Sections
- Content divided into clear sections with headings and paragraphs
- Structured using Bootstrap’s grid system
- Explains key concepts such as diversity, equity, and inclusion

### List of Tips
- A list of simple, practical actions users can take
- Styled using Bootstrap’s `list-group` component
- Easy to read and visually separated from other content

### Resource Links
- Links to external organisations and learning materials
- Styled as Bootstrap buttons for visibility and usability
- Encourages further independent learning

## Design Approach

- Mobile-first and responsive design
- Clear visual hierarchy
- Consistent spacing and alignment
- Simple colour palette for readability

---

## Accessibility Considerations

- Semantic HTML structure
- Clear headings and readable font sizes
- High contrast between text and background
- Logical content flow for screen readers
